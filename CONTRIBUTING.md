Install dependencies:

```shell
make install
```

Run checks:

```shell
make check
```

Run format:

```shell
make format
```

Automatically fix issues (format, lint):

```shell
make fix
```

Run docs:

```shell
make docs/serve
```

Review [Makefile](https://github.com/griptape-ai/griptape-nodes/blob/main/Makefile) for more commands.
